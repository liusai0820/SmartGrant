# src/ 架构说明

> ⚠️ **自指声明**：一旦我所属的文件夹有所变化，请更新我。

## 极简架构（3行）
- **app/** - Next.js 应用入口，包含页面、API路由、全局样式
- **components/** - React UI 组件库，实现评审面板、专家遴选等核心交互
- **lib/** - 核心服务层，封装 AI 调用、数据库操作、文档导出等

---

## 文件与子目录清单

| 名称 | 类型 | 地位 | 功能 |
|------|------|------|------|
| `app/` | 📁 目录 | 应用层 | Next.js App Router，路由与页面 |
| `components/` | 📁 目录 | 表现层 | React 组件，用户交互界面 |
| `lib/` | 📁 目录 | 服务层 | 业务逻辑、API封装、工具函数 |
| `types/` | 📁 目录 | 类型层 | TypeScript 类型定义 |
| `data/` | 📁 目录 | 数据层 | 静态数据与测试用例 |
